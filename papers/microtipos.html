Tipos de Microcontroladores

Antes de começar o artigo, recomendo que você leia um artigo ou o meu, que explica o que são microcontroladores.

Bem, dando uma resumida, microcontroladores são minicomputadores embalados em um chip de circuito integrado.

1. Arduino Uno

O Arduino Uno é um hardware de desenvolvimento baseado no microcontrolador ATmega328P da empresa Atmel, projetado para profissionais e estudantes criarem dispositivos eletrônicos.

A programação dele é baseada em C/C++ na Arduino IDE.

Ele possui uma memória flash de 32 KB (kilobytes), onde 0,5 KB é usada pelo bootloader. A memória flash é utilizada para armazenar o código do programa (o sketch) que você carrega no hardware.

Ele também possui uma memória SRAM (Static RAM) de 2 KB, que é usada para armazenar variáveis e dados temporários.

E sem contar a EEPROM (Electrically Erasable Programmable Read-Only Memory) de 1 KB, que pode ser usada para armazenar dados que precisam ser mantidos entre reinicializações da placa. A EEPROM é ideal para armazenar configurações ou outros dados que você não quer perder quando o hardware é desligado.

2. ESP8266

O ESP8266 é um microcontrolador que ganhou popularidade devido à sua capacidade de se conectar a redes Wi-Fi, tornando-o ideal para projetos de Internet das Coisas (IoT).

Ele possui um processador de 32 bits baseado no núcleo Tensilica Xtensa LX106, rodando a 80 MHz (podendo ser overclockado para 160 MHz), com 64 KB de RAM para dados e 96 KB de RAM para instruções.

Além disso, o ESP8266 conta com 4 MB de memória flash, usada para armazenar o firmware e programas.

É programável em diversas linguagens, como Lua, MicroPython e Arduino (C/C++).

3. Raspberry Pi Pico

O Raspberry Pi Pico é um microcontrolador lançado pela fundação Raspberry Pi. Ele utiliza o microcontrolador RP2040, projetado especificamente para esse hardware.

O RP2040 possui um processador dual-core ARM Cortex-M0+ rodando a 133 MHz, 264 KB de SRAM e 2 MB de memória flash embutida.

Uma das características mais notáveis do Raspberry Pi Pico é o suporte a uma vasta gama de interfaces de E/S, incluindo 26 pinos GPIO, interfaces I2C, SPI, UART, e ADCs, o que o torna uma escolha versátil para projetos variados.

O Raspberry Pi Pico pode ser programado utilizando MicroPython, C/C++, ou mesmo em linguagem assembly para operações mais avançadas.

4. STM32F103C8T6 (Blue Pill)

h

O STM32F103C8T6, popularmente conhecido como "Blue Pill", é um microcontrolador da família STM32, baseada no núcleo ARM Cortex-M3, com clock de até 72 MHz.

Ele possui 64 KB de memória flash, 20 KB de SRAM e diversas interfaces de comunicação, como USB, I2C, SPI, UART, e CAN.

O "Blue Pill" é bastante utilizado em projetos onde se precisa de maior poder de processamento em comparação com microcontroladores mais simples, como o Arduino.

Ele pode ser programado em C/C++ utilizando o ambiente de desenvolvimento STM32CubeIDE, ou através de ferramentas de código aberto como o PlatformIO.
