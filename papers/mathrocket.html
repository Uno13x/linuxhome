O que é MathRocket?

MathRocket é uma biblioteca Python projetada para facilitar a simulação e o cálculo de trajetórias de foguetes. Ela oferece métodos para calcular dinamicamente características essenciais de foguetes como massa, empuxo e tempo de queima, sendo ideal para análises de desempenho de lançamentos, estudos de física aplicada e simulações educacionais.

Instalando a Biblioteca

Para instalar a Biblioteca MathRocket, execute o seguinte comando no Terminal:

pip install MathRocket

Importando Módulos

Importe as seguintes classes da MathRocket no seu código Python:


from MathRocket.physics import Rocket
from MathRocket.trajectory import Trajectory
from MathRocket.visualization import Visualization

Simulando um Foguete

Aqui está um exemplo de como você pode simular e visualizar a trajetória de um foguete:


# Criar uma instância de Rocket
rocket = Rocket(mass=1000, thrust=500, burn_time=10)

# Simular a trajetória do foguete
trajectory = Trajectory.simulate(rocket)

# Visualizar a trajetória simulada
Visualization.plot_trajectory(trajectory.range, trajectory.flight_time, trajectory.details)
