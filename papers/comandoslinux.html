Comandos Básicos do Linux

Se você mudou para o Linux recentemente e não sabe como usar a linha de comando, irei te ensinar comandos básicos para começar.

Introdução à Linha de Comando

A linha de comando, também conhecida como terminal ou shell, é uma interface que permite ao usuário interagir com o sistema operacional através de comandos de texto. No Linux, a linha de comando é uma ferramenta poderosa e essencial para realizar diversas tarefas de forma eficiente.

Acessando o Terminal

Para abrir o terminal no Linux, você pode procurar por "Terminal" no menu de aplicativos ou usar o atalho de teclado Ctrl + Alt + T na maioria das distribuições.

Comandos Básicos

1. ls - Lista os arquivos e diretórios no diretório atual.

ls

2. cd - Muda o diretório atual.

cd /caminho/para/diretorio

3. pwd - Exibe o caminho completo do diretório atual.

pwd

4. mkdir - Cria um novo diretório.

mkdir nome_do_diretorio

5. rm - Remove arquivos ou diretórios.

rm nome_do_arquivo
rm -r nome_do_diretorio

6. cp - Copia arquivos ou diretórios.

cp arquivo_origem arquivo_destino
cp -r diretorio_origem diretorio_destino

7. mv - Move ou renomeia arquivos e diretórios.

mv arquivo_origem arquivo_destino

8. cat - Exibe o conteúdo de um arquivo.

cat nome_do_arquivo

9. nano ou vi - Editores de texto para modificar arquivos diretamente no terminal.

nano nome_do_arquivo
vi nome_do_arquivo

10. chmod - Altera as permissões de arquivos e diretórios.

chmod 755 nome_do_arquivo

Dicas Úteis

  • Utilize a tecla Tab para autocompletar nomes de arquivos e diretórios.
  • Use as setas para cima e para baixo para navegar pelos comandos anteriores.
  • Com o comando man seguido do nome de outro comando, você pode acessar o manual desse comando. Por exemplo: 
    man ls

Conclusão

Aprender a usar a linha de comando no Linux pode parecer desafiador no início, mas com a prática, você verá que ela oferece grande flexibilidade e controle sobre o sistema. Experimente os comandos básicos acima e explore mais funções à medida que se sentir confortável. Boa sorte!
